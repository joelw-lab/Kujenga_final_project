# Kujenga_final_project
Data-Driven Analysis of PROTAC Linker Properties for Improved Drug Discovery

# Overview

This project applies data science techniques to understand linker flexibility in PROTAC design for improved drug discovery.

# Techniques Used
Correlation analysis
Regression
Classification
Group comparison
Feature importance

# Key Findings
Molecular weight is the strongest driver of flexibility
Structural features regulate rigidity
Linker flexibility can be predicted

# Dataset
PROTAC linker dataset from PROTAC-DB

Open the notebook and run all cells.
